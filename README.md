## Descripción del Proyecto
Este proyecto consiste en un sistema modular en Python diseñado para gestionar las operaciones básicas de un restaurante, el cual permite la administración los menús, cantidades disponibles, precio y disponibilidad en lo que corresponde al producto, además permite administrar la información del cliente, en la que se incluye su nombre, teléfono y si es o no frecuente en visitar el restaurant.
